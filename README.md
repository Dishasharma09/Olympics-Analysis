# 🏅 Olympics Performance Analytics Dashboard

## 📌 Project Overview

The Olympics Performance Analytics Dashboard is a Business Intelligence project developed using **Python** and **Power BI**. The project transforms raw Olympic athlete data into meaningful insights through data cleaning, analysis, and interactive visualizations.

The dashboard helps analyze athlete participation, country performance, sport-wise achievements, medal trends, and factors influencing Olympic success.


## 🎯 Objectives

- Analyze athlete participation across Olympic years.
- Compare country-wise Olympic performance.
- Evaluate sport-wise medal distribution.
- Study athlete characteristics and medal success.
- Develop an interactive dashboard for decision-making.


## 🛠️ Technologies Used

- Pandas
- NumPy
- Matplotlib
- Power BI
- DAX
- Microsoft Excel

## 📈 Key Insights

- Olympic participation remained relatively stable across the selected years.
- Australia emerged as the best-performing country based on medal performance.
- Athletics demonstrated strong overall performance.
- Athlete success depends on multiple factors rather than physical characteristics alone.
- The overall medal rate is approximately 31%.
